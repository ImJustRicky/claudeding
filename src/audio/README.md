# Bundled Sound Effects

This folder contains the bundled sound effects for claudeding.

## Folder Structure

- `complete/` - Sounds for task completion
- `feedback/` - Sounds for when Claude needs input

## Adding Sounds

Place WAV files in the appropriate folder. The filename (without extension) becomes the sound name shown in the TUI.

Recommended naming: `gentle-chime.wav`, `soft-ding.wav`, etc.

## Sources for Free Sounds

- [Pixabay](https://pixabay.com/sound-effects/search/ding/) - CC0/royalty-free
- [Mixkit](https://mixkit.co/free-sound-effects/notification/) - Royalty-free
- [Freesound](https://freesound.org/browse/tags/ding/) - Various CC licenses

Note: Make sure sounds are short (< 2 seconds) and not jarring.
